Embed Google Apps Script (Web App)

Penting: Link yang Anda kirim /macros/library/... adalah Apps Script Library dan tidak bisa di-embed.
Yang bisa di-embed hanya URL Web App deployment dengan format https://script.google.com/macros/s/.../exec.
Jika iframe blank/blocked, biasanya karena setelan akses Web App, kebijakan browser, atau web app menolak embedding.

Jika iframe gagal load: pastikan Web App di-deploy dengan akses yang benar (mis. Anyone with the link) dan Anda memakai URL /exec dari deployment terbaru.
